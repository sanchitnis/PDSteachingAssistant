# Advanced C Programming with Generative AI — Course Syllabus & Exercise Materials

---

## 1. Course Outcomes (COs)

| CO | Description |
|---|---|
| **CO1** | Develop modular and reusable code using functions (call by value and call by reference), arrays and structures using illustrative programs and test-driven development. |
| **CO2** | Apply modular programming to manage complex data collections using Arrays of Structures through illustrative programs and test-driven development. |
| **CO3** | Develop modular and reusable code using pointers and dynamic memory allocation using illustrative programs and test-driven development. |
| **CO4** | Develop modular and reusable code utilizing Strings by implementing user-defined logic for processing text-based data through illustrative programs and test-driven development. |

---

## 2. Course Content (Theory & Concepts)

### 1. C Fundamentals
Quick review of basic C programming concepts: arrays, structures, array of structures, and functions. Function prototypes, definition, and scope (local vs. global). Parameter passing mechanisms — call by value and call by reference through a modular programming approach. Error handling.

### 2. Array of Structures
Passing arrays to functions as parameters. Operators to access structure members. Passing structures to functions as parameters. Passing array of structures to functions as parameters.

### 3. Pointers
Understanding pointers and dynamic memory management using `malloc`, `calloc`, `realloc`, and `free`. Pointers and arrays. Handling dangling pointers and preventing memory leaks. Self-referential structures and singly linked lists as an application of pointers.

### 4. Text Processing
Reading and manipulating strings using user-defined functions and pointers.

### 5. File Handling
File handling functions for various operations:

| Operation | Functions |
|---|---|
| Opening a file | `fopen()` |
| Reading from a file | `fread()`, `fgets()` |
| Writing to a file | `fwrite()`, `fprintf()` |
| Closing a file | `fclose()` |
| Error checking | `feof()`, `ferror()` |
| Modes of operation | `r`, `w`, `a`, and binary modes |

### 6. Mini-Project
Mini-project using AI tools and prompt engineering, utilising all concepts covered above.
